# tiny-img

利用tinypng批量压缩文件夹中的图片

### Installation

```bash
$ npm install tiny-img -g (国内网络原因，暂未发布)
```

### Usage
In your current path

```bash
$ tiny-img key
```
```bash
$ tiny-img -key key
```
```bash
$ tiny-img --key key
```

### Where will get the key

See the [tinypng.com](https://tinypng.com/developers).
