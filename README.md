# tiny-img

Use tinypng to bulk compress the images in the folder

利用tinypng批量压缩文件夹中的图片

### Installation

```bash
$ npm install tiny-img -g
```

### Usage
In your current path

```bash
$ tiny-img key
```
```bash
$ tiny-img -key key
```
```bash
$ tiny-img --key key
```

### Where will get the key

See the [tinypng.com](https://tinypng.com/developers).
